Container tags

A container in HTML is set of tags used to wraps other HTML elements.It's good to wrap multiple tags in an overall containers.
The main containers are- Divs and Spans

HTML has 2 basic main ways to display elements:-

  • Blocks
  • Inline

Blocks
- They are HTML elements which take entire width of the page.

Paragraph 1

Paragraph 2


Here we have not used <br> and still we are getting these both paragraphs in different lines (even if so much space is available), this means they occupy a complete block, and hence are called "block display" or "Div containers"
Inline
They take only as much space as required
example:- YouTube Google

Here we have not written any <br> and hence they both appear together and hence are called "inline display" or "Spans containers"

NOTE:-

to write less than (<) and greater than sign (>), we can use &lt; and &gt; respectively

Examples:-

Code

<div> div1 </div>
<div> div2 </div>

Output:-

div1
div2

<span> span1 </span>
<span> span2 </span>

Output:-

span1 span2
